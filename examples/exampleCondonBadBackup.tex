Note that $\mathsf{f_2}$ is not a target anymore as in \cnf  there is exactly one target and one sink. Therefor $\mathsf{f_2}$ has an action going to $\target$. However as it is a state with one action and all must have exactly two, we have to add an action going to the sink $\mathrm{\zeroSink}$ as in $\mathsf{s_0}$. All the $\Circ$-states with only one action must have an action going to $\target$ but we omitted so that the figure is easier readible.}
